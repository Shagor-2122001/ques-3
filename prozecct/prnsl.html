Become a web Developer

Web Developers are responsible for creating websites that are both functionally and aesthetically pleasing to the user. The typical work activities of a Web Developer include back-end tasks such as developing databases, writing and testing code for website functions, troubleshooting code in existing features and brainstorming new features based on user feedback. Their work activities also include front-end tasks like choosing color palettes, font pairings, and making other aesthetic-based decisions.
